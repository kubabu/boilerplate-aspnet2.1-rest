# boilerplate-aspnet2.1-rest

Bolerplate .NET Core 2.1 REST to work with Angular >=5

## Features
- EF Core with PostgreSQL 
- SSL with JWT
- role based controller authorization
- SignalR realtime communication with client
- works over nginx on Linux server

## Build, run tests
Open soluition with Visual Studio, or 

	cd WebApi 

and follow [instructions](WebApi/README.md).
